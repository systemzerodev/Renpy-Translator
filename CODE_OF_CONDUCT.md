Code of Conduct

Our Pledge

We are committed to making participation in this project a respectful and harassment-free experience for everyone.

---

🤝 Expected Behavior

- Be respectful and inclusive
- Accept constructive feedback
- Focus on what is best for the community
- Be patient with beginners

---

🚫 Unacceptable Behavior

- Harassment or discrimination
- Offensive comments
- Personal attacks
- Spamming or trolling

---

⚖️ Enforcement

Project maintainers are responsible for enforcing this code.

Violations may result in:

- Warning
- Temporary ban
- Permanent ban

---

📬 Reporting Issues

If you experience or witness unacceptable behavior, please report it via:

- GitHub Issues
- Direct message to maintainer

---

❤️ Final Note

We want this project to be a safe and welcoming place for everyone.
