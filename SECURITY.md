Security Policy

🔒 Supported Versions

Currently, only the latest version of RenPy Translator is supported with security updates.

---

🚨 Reporting a Vulnerability

If you discover a security vulnerability, please:

1. DO NOT open a public issue
2. Contact the maintainer privately
3. Provide detailed information

---

📧 Contact

You can report security issues via:

- GitHub private message
- Email (if provided)

---

⏱ Response Time

We aim to respond within:

- 48 hours for initial response
- 3–7 days for investigation

---

🙏 Responsible Disclosure

We appreciate responsible disclosure and will credit contributors when appropriate.
